## 远程同步
(1) 取回远程仓库的变化，并与本地分支合并。  
```git
git pull [remote] [branch]
```
(2) 上传本地指定分支到远程仓库。  
```git
git push [remote] [branch]
```
(3) 强行推送当前分支到远程仓库，即使有冲突。  
```git
git push [remote] --force
```
(4) 推送所有分支到远程仓库。  
```git
git push [remote] --all
```

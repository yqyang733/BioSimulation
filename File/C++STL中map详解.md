## 简介
是一个关联容器，以键值对存储数据，其类型可以自定义，每个关键字在map中只能出现一次，关键字不能修改，值可以修改。

## 用法
### 头文件
```cpp
#include<map>
```

### 定义
```cpp
map<string, int> my_map;
```

### 基本方法
* my_map.count(key)
**简介：** 查看key元素是否在容器内，如果在返回1，如果不在返回0。
* my_map.emplace(key, element)
**简介：** 将键及其元素插入到映射容器中。如果多次放置同一个键，则映射仅存储第一个元素，因为映射是不存储多个相同值的键的容器。
**参数：**
（1）key：指定要在多图容器中插入的键。
（2）element：指定要插入Map容器的键对应的值。
* my_map[key]
**简介：** 取出键对应的值。

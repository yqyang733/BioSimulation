## 1. 简介

在对分子对接结果或者晶体结构的蛋白配体复合物进行分析的时候，蛋白配体二维相互作用图往往能更加直观的显现出蛋白配体间各种类型的相互作用力。本篇文章介绍使用Schrodinger的 2D Sketcher 模块绘制蛋白配体2D相互作用图。

---

## 2. 绘图

（1）先将对接结果或者晶体结构复合物导入Schrodinger中。

（2）如PPT中图所示，在搜索界面调出2D Sketcher界面，即可自动完成绘制。

---

## 3. 调整图片参数导出图片

（1）如PPT中图片所示。可以调整下列参数对图片显示进行调整。

Ligand：指定显示相互作用的配体。

Cutoff：指定截取配体周围一定范围内的残基显示相互作用。

其他参数自行调整尝试即可。

（2）View -> LID Legend，显示互作类型标签。如PPT中图片所示。

（3）File -> Screenshot，保存图片即可。
